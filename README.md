# Sniffer

## Запуск
Для запуска используйте команду 
```
make
```

А затем запустите бинарный файл с помощью 
```
./build/main [mode] [source]
```

Существует 3 вида чтения:
+ file_mode - чтение указанного файла
+ directory_mode - чтение всех *.pcap файлов из директории
+ live_mode - чтение с интерфейса

Пример запуска:
```
./build/main file_mode ./test.pcap
```

Для пересборки:
```
make rebuild
```

## Описание программы
Программа анализирует сетевой трафик и распределяет его по обработчикам. Программа отделяет FTP команды (порт 21) от FTP данных. Также программа способна находить корректные TCP соединения и собирать их.
